# 💰 Sistema Financeiro

Sistema financeiro em desenvolvimento, baseado no curso do Prof. Valdir Ferreira (Dev Net Core), com foco na aplicação prática de conceitos modernos de desenvolvimento full stack e arquitetura em camadas Utiliza C# (.NET) no back-end, Angular no front-end e SQL Server como banco de dados. Focado na prática de desenvolvimento full stack e construção de APIs.


🚀 Tecnologias utilizadas
- Back-end: C# com .NET 7 (API REST)
- Front-end: Angular 15
- Banco de dados: SQL Server
- Arquitetura: DDD (Domain-Driven Design)

📌 Funcionalidades
- Cadastro de usuários
- Controle de receitas e despesas
- Dashboard financeiro
- Autenticação e autorização
- Validações de domínio
- Integração entre API e front-end

📂 Estrutura do Projeto
```
/backend
  ├──  Domain
  ├──  Application
  ├──  Infrastructure
  └──  API

/frontend
  └──  Angular App
```
  
  
⚙️ Como executar o projeto
🔧 Back-end (.NET)

- cd backend
- dotnet restore
- dotnet run
  
💻 Front-end (Angular)
- cd frontend
- npm install
- ng serve
  
🎯 Objetivo

Projeto desenvolvido com fins de estudo, visando consolidar conhecimentos em desenvolvimento full stack, arquitetura DDD, criação de APIs REST e integração com aplicações modernas.
